🛒 SunCart - Seasonal B2C eCommerce Platform
SunCart is a modern and responsive seasonal eCommerce platform designed specifically for selling summer collections and seasonal products. This project prioritizes performance and user experience, built with a robust tech stack to ensure scalability and ease of use.

🚀 Key Features
Dynamic Hero Slider: Features an engaging summer sale banner and a "Hot Deals 🔥" slider to highlight offers.

Authentication: Secure login and registration powered by Better Auth, supporting both Email/Password and Google Login.

Protected Routes: Product details and user profile pages are restricted to authenticated users only.

Product Discovery: Displays popular products with ratings, pricing, and category filtering (e.g., Summer Outfits, Skincare).

User Profile Management: Includes a dedicated "My Profile" page where users can view their data and update their name or image.

Responsive Layout: Fully optimized for mobile, tablet, and desktop views.

🛠 Tech Stack
Framework: Next.js (App Router with Turbopack).

UI Library: HeroUI (formerly NextUI).

Styling: Tailwind CSS.

Authentication: Better Auth.

Database: MongoDB.

📂 Project Structure
src/: Main source code including components and app router logic.

public/: Static assets, images, and icons.

lib/: Authentication configuration and database utility functions.
